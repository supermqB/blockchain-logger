北方健康区块链 - 数据开放日志上链项目

请使用 /addlog 来将日志上链。

/addlog 
    参数示例：
    {
        "userId": "bcosuser",
        "keyURL": "/test/add",
        "content": "rawcontent"
    }

请使用 /verifylog 来将检查日志是否被篡改。

/verifylog 
    参数示例：
    {
        "logId": "40c1baa13514ed416044c2e9a1db666b",
        "content":"rawcontent"
    }
